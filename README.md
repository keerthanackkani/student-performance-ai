# 🎓 Student Performance Prediction using AI

## 📌 Project Overview

This project is a Machine Learning model that predicts a student's academic performance (Poor, Average, Good) based on study habits and lifestyle factors.

---

## 🧠 Problem Statement

Student performance depends on multiple factors such as study hours, attendance, sleep, and assignment completion. This project uses AI to analyze these factors and predict performance levels.

---

## 🚀 Features

* Predicts student performance using Machine Learning
* Data analysis using Python
* Simple and beginner-friendly model
* Fast prediction system
* Easy to understand and modify

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## 📊 Dataset Features

* Study Hours
* Attendance Percentage
* Sleep Hours
* Assignments Completed
* Performance Label (Poor / Average / Good)

---

## 🤖 Machine Learning Model

* Algorithm used: Decision Tree Classifier
* Supervised Learning approach
* Trained on sample student dataset

---

## 📈 Output

The model predicts:

* Poor
* Average
* Good

---

## ▶️ How to Run

1. Clone this repository
2. Install required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Run the Python file:

   ```
   python main.py
   ```
